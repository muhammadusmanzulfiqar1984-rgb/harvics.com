# 🎯 CRM Roles - Quick Reference Card

## 👤 ROLE → WHAT THEY SEE

### 🛡️ SUPER ADMIN
```
✅ Sees EVERYTHING
   • All Distributors (all countries)
   • All Suppliers (all countries)  
   • All Company Data
   • Global Reports
```

### 🏢 COMPANY ADMIN  
```
✅ Sees All Company + All Distributors + All Suppliers
   • Can CREATE distributors
   • Can CREATE suppliers
   • Company-wide data
```

### 📈 SALES MANAGER
```
✅ Can CREATE distributors ⭐
✅ Can ANALYZE distributors ⭐
✅ Sees all distributors
   ❌ Cannot see suppliers
```

### 📦 DISTRIBUTOR
```
✅ Sees ONLY their own data
   • Their orders
   • Their inventory
   • Their customers
```

### 🚚 SUPPLIER
```
✅ Sees ONLY their own data
   • Their purchase orders
   • Their inventory
   • Their shipments
```
